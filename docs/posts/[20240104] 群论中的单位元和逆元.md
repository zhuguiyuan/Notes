# 群论中的单位元和逆元

今天开组会的时候同门谈到了一个有趣的问题，记录如下，顺便测试一下数学公式的展示效果。

假设我们有一个集合，在上面有一个左右都结合的二元运算符（函数）$f$，那么：

1. 假设有一个左单位元 $e_0$ 使得 $(\forall x) e_0x = x$，
并且每个元素也有左逆 $(\forall a) (\exists a^{-1}) a^{-1} a = e_0$，
证明可以导出右单位元 $(\forall x) xe_1 = x$
和右逆 $(\forall a) (\exists a^{-1}) a a^{-1} = e_1$，
然后顺便 $e_0 = e_1$。
2. 假设在这个集合和运算符下有一个左单位元 $e_0$（同上），
每个元素也都存在右逆（同上），
举一个反例说明这两个条件不能导出存在右单位元（同上）和左逆（同上）。

<!-- more -->

对 1，先证明有右逆：
$a a^{-1} = e a a^{-1} = ((a^{-1}) a^{-1}) (a a^{-1}) = e$；
然后右单位元就简单了：$a e = a (a^{-1} a) = e a = a$。

对 2，反例是 $\mathbb{Q} - \{0\}$ 上定义的 $f(x, y) = |x| y$。
$1$ 是左单位元，对于 $x$，右逆是 $y = |1/x|$。
不存在右单位元，因为对于这个集合中的 $x$ 和 $-x$，
没办法同时满足 $|x| y = x \land |-x| y = -x$。
同样也不存在左逆，对于 $y$ 和 $-y$，$|x_1| y = e \land - |x_2| y = e$，两个式子减一下得到 $(|x_1| + |x_2|) y = 0$，给定的集合里面不存在这样的 $x_1$，$x_2$ 和 $y$。
